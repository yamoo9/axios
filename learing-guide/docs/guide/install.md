# 설치

Axios를 사용하려면 먼저 설치를 진행해야 합니다.

### NPM 설치

Node.js 개발 환경에서 설치하는 방법입니다.

```bash
$ npm install axios
# 또는 yarn add axios
```

### CDN 설치

브라우저 환경에서 설치/사용하는 방법입니다.

```html
<script src="https://unpkg.com/axios/dist/axios.min.js"></script>
```
